# Security Policy

## Supported Versions
Only the latest version of HoxCore is currently supported for security updates.

| Version | Supported          |
| ------- | ------------------ |
| 0.1.x   | :white_check_mark: |

## Reporting a Vulnerability
If you discover a security vulnerability within HoxCore, please do **not** open a public issue. Instead, send an email to **salvadec@live.com.mx**.

We will acknowledge your report within 48 hours and provide an estimated timeline for a fix. Please include:
- A description of the vulnerability.
- A proof-of-concept or steps to reproduce.
- Potential impact.
